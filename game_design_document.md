# SLEEPWALKER — Game Design Document

**Версия:** 0.5  
**Статус:** В разработке  
**Жанр:** Комедийный социальный стелс  
**Платформы:** PC (Steam), консоли (перспектива)

---

## 1. EXECUTIVE SUMMARY

### 1.1 Elevator Pitch

Стелс-игра, где ты не убиваешь охранников — ты прячешься от бабушек, туристов и камер смартфонов. Потому что ты голый. Потому что ты лунатик.

### 1.2 Уникальное торговое предложение (USP)

| Что есть у конкурентов | Что делаем мы |
|------------------------|---------------|
| Стелс = насилие и убийства | Стелс = социальный стыд |
| Серьёзный тон | Комедия в духе Mr. Bean |
| Фиксированные уровни | Процедурные ситуации |
| Враги патрулируют | NPC живут своей жизнью |

**Формула:** Hitman (социальный стелс) + Untitled Goose Game (комедия) + процедурная генерация

### 1.3 Целевая аудитория

- **Первичная:** Любители инди-игр 18-35, ценящие юмор и нестандартные механики
- **Вторичная:** Стримеры и контент-криейторы (игра генерирует смешные моменты)
- **Третичная:** Казуальные игроки, фанаты Untitled Goose Game, Human Fall Flat

### 1.4 Референсы

| Игра | Что берём |
|------|-----------|
| Hitman | Социальный стелс, маскировка, песочница |
| Untitled Goose Game | Тон, комедия через геймплей |
| Human Fall Flat | Физика, абсурд, кооп-потенциал |
| Mr. Bean (не игра) | Ситуационный юмор без слов |

## 2. CORE GAMEPLAY

### 2.1 Философия дизайна

**Главный вопрос игрока:** "Как я выкручусь из этой ситуации?"

Каждая сессия — это импровизационная комедия, где игрок сам создаёт смешные моменты через свои решения. Игра не рассказывает шутки — она создаёт условия для их возникновения.

### 2.2 Core Loop

```
│       │              │                         │       │
```

### 2.3 Фазы игровой сессии

#### Фаза 1: Пробуждение (5-10 сек)
- Экран темнеет
- Звук окружения нарастает
- Камера показывает ситуацию
- Игрок оценивает: где я? что вокруг? какие угрозы?

#### Фаза 2: Выживание (3-10 мин)
- Основной геймплей
- Поиск укрытия/одежды
- Взаимодействие с NPC
- Стелс и импровизация

#### Фаза 3: Эвакуация (1-3 мин)
- Добраться до безопасной зоны
- Финальный рывок или спокойный выход

### 2.4 Условия завершения уровня

| Исход | Условие | Награда |
|-------|---------|---------|
| **Идеальный** | Полная одежда + никто не видел | 3 звезды + бонус |
| **Успех** | Прикрыт + достиг выхода | 2 звезды |
| **Выживание** | Достиг выхода любой ценой | 1 звезда |
| **Провал** | Арест / вирусный мем / потеря сознания от стыда | 0 звёзд, рестарт |

### 2.5 Шкала "Позора" (Shame Meter)

Центральная механика, заменяющая "здоровье":

```
[░░░░░░░░░░] 0%   — Полностью одет, инкогнито
[████░░░░░░] 40%  — Замечен, неловкие взгляды
[███████░░░] 70%  — Снимают на телефоны
[██████████] 100% — ПРОВАЛ (мем, арест, обморок)
```

**Что повышает Shame:**
- Быть замеченным без одежды (+10-30%)
- Попасть на камеру (+15%)
- Попасть на телефон NPC (+25%)
- Неудачное враньё (+10%)
- Крики NPC (+5% за каждого свидетеля)

**Что понижает Shame:**
- Найти одежду (-20-50%)
- Успешно спрятаться (-5%)
- Убедить NPC что всё ок (-10%)
- Время без обнаружения (-1%/10сек)

## 3. МЕХАНИКИ

### 3.1 Система обнаружения

#### 3.1.1 Состояния NPC по отношению к игроку

```
UNAWARE ──▶ SUSPICIOUS ──▶ ALERTED ──▶ REACTING
   │             │              │            │
   │             │              │            ▼
   │             │              │      [Тип реакции]
   │             │              │       /    |    \
   │             ▼              ▼    Помощь Паника Агрессия
   │        Исследует      Приближается
   │             │              │
   └─────────────┴──────────────┘
            (потеря интереса)
```

#### 3.1.2 Параметры обнаружения

| Параметр | Описание | Диапазон |
|----------|----------|----------|
| **Дистанция** | Максимальная дальность обнаружения | 5-20м |
| **Угол зрения** | Конус видимости NPC | 60°-120° |
| **Скорость обнаружения** | Как быстро заполняется "подозрение" | 0.5-3 сек |
| **Внимательность** | Множитель скорости | 0.5x-2x |

#### 3.1.3 Модификаторы видимости игрока

| Фактор | Модификатор |
|--------|-------------|
| Полностью голый | 1.0x (базовый) |
| Импровизированное прикрытие | 0.7x |
| Частичная одежда | 0.4x |
| Полная одежда | 0.1x |
| В тени/укрытии | 0.3x |
| Движение (бег) | 1.5x |
| Движение (красться) | 0.6x |
| NPC отвлечён | 0.5x |

**Формула:**
```
Шанс обнаружения = Базовый шанс × Видимость × Внимательность NPC × (1 / Дистанция²)
```

### 3.2 Система одежды и прикрытия

#### 3.2.1 Категории предметов

| Категория | Примеры | Эффект на Shame | Эффект на видимость | Прочность |
|-----------|---------|-----------------|---------------------|-----------|
| **Нет** | — | +2%/сек на виду | 1.0x | — |
| **Импровизация** | Газета, пакет, поднос | Стоп деградации | 0.7x | 1 использование |
| **Временное** | Полотенце, занавеска | -20% | 0.5x | 30 сек активности |
| **Частичное** | Шорты, футболка | -35% | 0.4x | Постоянно |
| **Полное** | Комплект одежды | -50% | 0.2x | Постоянно |
| **Маскировка** | Униформа, костюм | -50% + спец.доступ | 0.1x | До разоблачения |

#### 3.2.2 Специальные свойства одежды

- **Униформа охранника:** Доступ в служебные помещения
- **Халат врача/работника:** NPC меньше удивляются
- **Смешной костюм:** Shame не растёт, но все смотрят
- **Маска:** Защита от распознавания на камерах

### 3.3 Система взаимодействия с NPC

#### 3.3.1 Типы взаимодействий

| Действие | Условие | Результат при успехе | Результат при провале |
|----------|---------|---------------------|----------------------|
| **Просить помощь** | NPC с высокой эмпатией | Даёт предмет/подсказку | Игнор или паника |
| **Врать** | Проверка Харизмы | NPC отворачивается | Подозрение +50% |
| **Отвлечь** | Есть предмет для броска | NPC идёт к источнику звука | NPC замечает игрока |
| **Прятаться в толпе** | 3+ NPC рядом | Временная невидимость | Кто-то замечает |
| **Шокировать** | — | NPC в ступоре 3 сек | Крик, Shame +30% |

#### 3.3.2 Социальные проверки

```
Успех = (Харизма игрока + Бонус одежды + d20) > (Подозрительность NPC × 10 + Порог)
```

Пороги:
- Лёгкая ложь: 10
- Убедительная история: 15
- Абсурдное объяснение: 20
- "Я с телевидения": 25

### 3.4 Система импровизации

#### 3.4.1 Использование окружения

| Объект | Использование | Навык |
|--------|---------------|-------|
| Тележка уборщицы | Укрытие, транспорт | Базовый |
| Картонная коробка | Классическое укрытие | Базовый |
| Манекен | Притвориться манекеном | Импровизация 1 |
| Фонтан | Укрытие в воде | Базовый |
| Занавески | Срывать для одежды | Импровизация 1 |
| Пожарная сигнализация | Эвакуация всех NPC | Импровизация 2 |
| Электрощиток | Отключить свет | Импровизация 2 |

#### 3.4.2 Комбинации предметов (продвинутый геймплей)

- Газета + Скотч = Импровизированная юбка (дольше держится)
- Мусорный пакет + Верёвка = Пончо
- Два подноса = Щиты спереди и сзади

## 6. НАРРАТИВ

### 6.1 Сеттинг

Современный город. Реалистичный мир, абсурдные ситуации. Никакой магии или фантастики — весь юмор из жизненных ситуаций, доведённых до крайности.

### 6.2 История (минимальная, для контекста)

**Главный герой** — обычный человек (офисный работник? художник? на выбор), у которого после стресса развилась тяжёлая форма сомнамбулизма. Врачи не помогают. Снотворное не работает. Каждую ночь — рулетка.

**Цель:** Найти способ контролировать болезнь. Или хотя бы научиться справляться с последствиями.

**Структура (если делаем кампанию):**
1. Акт 1: Диагноз — обучение, простые ситуации
2. Акт 2: Эскалация — ситуации всё абсурднее
3. Акт 3: Принятие — финальные испытания, "босс-уровни"

### 6.3 Тон

- **Юмор:** Ситуационный, физический, без пошлости
- **Не смеёмся над:** болезнью, телами, уязвимыми группами
- **Смеёмся над:** абсурдом ситуации, реакциями NPC, изобретательными решениями игрока

## 6.5 СОЦИАЛЬНЫЙ МЕССЕДЖ: ОСВЕДОМЛЁННОСТЬ О СОМНАМБУЛИЗМЕ

### 6.5.1 Почему это важно

Сомнамбулизм — реальное расстройство, затрагивающее ~1.5% взрослых. Исследования показывают:
- Повышенная частота депрессии, тревожности, сниженное качество жизни
- 58% пациентов испытывают агрессивное поведение во сне
- Эпизоды вызывают острое смущение и разрушают отношения
- Расстройство недодиагностировано — люди стесняются обращаться за помощью
- Общество часто воспринимает это как "смешную" проблему

**Наша возможность:** Через игру показать проблему изнутри, вызвать эмпатию и изменить отношение общества.

### 6.5.2 Принципы подхода

| Делаем | Не делаем |
|--------|-----------|
| Смеёмся над ситуациями | Смеёмся над болезнью |
| Показываем уязвимость героя | Превращаем болезнь в суперсилу |
| Даём моменты человечности | Постоянный фарс без передышки |
| Образовываем ненавязчиво | Читаем лекции игроку |
| Показываем разные реакции NPC | Все NPC реагируют одинаково негативно |

**Референс тона:** Celeste (тревожность через геймплей) + Untitled Goose Game (юмор)

### 6.5.3 Интеграция в геймплей

#### Эмпатичные NPC
Не все NPC реагируют негативно. Некоторые:
- Предлагают помощь без осуждения
- Узнают симптомы ("Моя сестра тоже так делает")
- Дают полотенце/одежду и отворачиваются

**Механика:** Взаимодействие с такими NPC снижает Shame больше, чем обычно. Игрок запоминает, что понимание помогает.

#### "Тихие моменты"
Между уровнями — короткие сцены в безопасной зоне:
- ГГ смотрит на замки на дверях, которые сам установил
- Звонок от близкого человека с поддержкой
- Записка от соседа: "Видел тебя ночью. Всё ок. Если что — стучи"

Эти моменты не мешают геймплею, но добавляют глубину.

#### Дневник/записки
Коллекционные предметы — заметки ГГ о своём состоянии:
- Реальные факты о сомнамбулизме (ненавязчиво)
- Эмоциональные записи о страхах и надеждах
- Советы от врачей

### 6.5.4 Нарративная арка

**Акт 1: Отрицание**
- ГГ пытается скрыть проблему
- Изолируется от людей
- Shame механика особенно жёсткая

**Акт 2: Борьба**
- Попытки контролировать (будильники, замки, снотворное)
- Некоторые NPC показывают понимание
- Появляются "союзники"

**Акт 3: Принятие**
- ГГ принимает состояние как часть себя
- Находит способы справляться
- Финальный уровень: просыпается в публичном месте, но реагирует спокойнее
- Shame механика менее карающая (прогресс персонажа = прогресс принятия)

### 6.5.5 Партнёрство и awareness

**Потенциальные партнёры:**
- American Academy of Sleep Medicine (AASM)
- Sleep Foundation
- Project Sleep (у них есть горячая линия поддержки)

**Что можем сделать:**
- Экран с информацией о расстройстве в главном меню (опционально)
- Ссылки на ресурсы поддержки
- % от продаж на исследования (маркетинговый ход + реальная помощь)
- Консультации с людьми, живущими с сомнамбулизмом — на этапе разработки

### 6.5.6 Что это даёт проекту

| Аспект | Влияние |
|--------|---------|
| **Глубина** | Игра не просто "прикольная", а со смыслом |
| **Уникальность** | Никто так не делал в жанре комедийного стелса |
| **PR** | Социальный месседж = пресса, награды (Games for Change) |
| **Аудитория** | Люди с расстройством и их близкие = лояльные игроки |
| **Долговечность** | Игра с месседжем запоминается дольше |

### 6.5.7 Риски и митигация

| Риск | Митигация |
|------|-----------|
| Становится слишком серьёзно | Чёткий баланс: 80% юмор, 20% глубина |
| Оскорбление людей с расстройством | Консультации на этапе разработки, sensitivity читка |
| "Морализаторство" | Вся информация опциональна, не блокирует геймплей |
| Несовместимость тона | Плейтесты с фокусом на эмоциональный отклик |

| Время прохождения | 30-60 минут |

### 9.2 Команда MVP

| Роль | Кол-во | Обязанности |
|------|--------|-------------|
| Геймдизайнер | 1 | Баланс, уровни, документация |
| Программист | 1-2 | Системы, AI, UI |
| 3D-художник | 1 | Персонажи, окружение |
| Аниматор | 0.5 | Анимации персонажей (может быть тот же художник) |
| Левел-дизайнер | 1 | Локация, расстановка |
| Звукорежиссёр | 0.5 | Аутсорс или парт-тайм |

**Итого: 4-6 человек**

### 9.3 Timeline MVP


### 9.4 Риски

| Риск | Вероятность | Импакт | Митигация |
|------|-------------|--------|-----------|
| AI NPC слишком тупой/умный | Высокая | Высокий | Ранние плейтесты, итерации |
| Юмор не работает | Средняя | Критический | Тестирование на аудитории |
| Scope creep | Высокая | Высокий | Жёсткие границы MVP |
| Баланс сложности | Высокая | Средний | Система адаптивной сложности |
| Однообразие геймплея | Средняя | Высокий | Разнообразие ситуаций в рамках одной локации |
