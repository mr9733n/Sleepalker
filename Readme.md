# Sleepwalker 

    A comedy stealth game about a sleepwalker who wakes up every night in absurd situations.

## Для инвесторов/паблишеров:
```text
Sleepwalker — комедийный стелс о лунатике, который каждую ночь просыпается в абсурдных ситуациях.
Но за юмором — реальная проблема: 1.5% взрослых живут с сомнамбулизмом. Они испытывают стыд, тревогу, разрушенные отношения. Общество смеётся над тем, чего не понимает.
Мы создаём игру, которая даёт прожить этот опыт изнутри. Смеяться над ситуациями — но сочувствовать человеку. Как Celeste сделала для тревожности, мы делаем для сомнамбулизма.
Комедия с эмпатией. Стелс со смыслом.
```

## Для прессы/соцсетей:
```text
Проснулся голым в лифте отеля. Лифт едет вниз.
Это не кошмар — это вторник для 1.5% взрослых с сомнамбулизмом.
Sleepwalker — игра о том, каково это на самом деле. Смешно? Да. Но когда в следующий раз увидишь человека в странной ситуации — может, вместо смеха предложишь полотенце.
```
